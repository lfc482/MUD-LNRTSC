# Model checkpoints

This folder contains trained model checkpoint files for the six leave-one-well-out cross-validation folds.

The checkpoint files are:

- `proposed_test_well_1_best.pth`
- `proposed_test_well_2_best.pth`
- `proposed_test_well_3_best.pth`
- `proposed_test_well_4_best.pth`
- `proposed_test_well_5_best.pth`
- `proposed_test_well_6_best.pth`

These files contain trained model parameters only. The raw field drilling data and model output results are not included due to confidentiality restrictions.
